# Backward_bifurcations_in_spatial_replicator_models

This repository contains code for the numerical results used in the paper:

**Freire, T., Gjini, E., & Madec, S. (2026).**
*Backward bifurcations in spatial replicator models: when invasion criteria fail to predict coexistence*
https://doi.org/XX.XXXX/XXXXXX

---

## Repository Contents

### `spatial_replicator_simulations.ipynb`

Python notebook containing the numerical simulations for the spatial two-species replicator model.

### `hetero_ab_additive.pkl`

Pickle file containing the simulation data used for the final figure of the paper.

---

## Model

The code focuses on a spatial two-species replicator equation with diffusion and advection. The main goal is to study how spatial heterogeneity can produce backward bifurcations, allowing coexistence in regimes where classical invasion criteria suggest exclusion.
